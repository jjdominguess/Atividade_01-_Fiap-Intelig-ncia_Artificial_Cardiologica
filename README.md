## 📜 Descrição
  - Esta atividade visa trazer dados sobre uma conjunto de pessoas, dados esses que serão usados para traçar paralelos com comorbidades cardiológicas.
  Consideramos variáveis importantes idade, colesterol, pressão arterial e dores no peito. 
  Inicialmente idade é importante por fatos que quanto mais idade se tem o orgânismo produz capiladirade de veias fazendo com que a taxa de sobrevivência a um ataque cardíaco seja mais alta, um pessoa que seja mais jovem também possui vantagem por possuir uma capacidade rítmica cardiaca maior. O colestorol sendo alto pode obstruir veias ou até artérias e causar sobrecarga no coração, sendo assim também aumentando a pressão arterial, outrossim o colesterol e pressão arterial dizem muito em relação a alimentação do indivíduo, não sendo mandatório, pois condições psicológicas podem trazer alterações, mas não tanto quanto uma alimentação desbalanceada.

- Data set utilizado para compor a planilha que foi analisada por nós foi [Kaggle](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset), o mesmo é um dataset real e 1978, da região de Cleveland(EUA), Hungria e Suíça.
